# FIRNADIQ GRAPHICS Website

Premium responsive website for FIRNADIQ GRAPHICS, built with Next.js 15, React, TypeScript, Tailwind CSS, Framer Motion, dark/light mode, SEO metadata, sitemap, robots, and Firebase-ready student login.

## Run locally

```bash
npm install
npm run dev
```

## Firebase setup

Copy `.env.example` to `.env.local` and fill the `NEXT_PUBLIC_FIREBASE_*` values to enable Firebase authentication for the student login page.
